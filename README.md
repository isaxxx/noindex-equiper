# noindex-equiper

Add noindex to HTML(.html or .htm).

[![MIT License](http://img.shields.io/badge/license-MIT-blue.svg?style=flat)](LICENSE)

## Usage

### Step1

```bash
$ npm install
```

### Step2

Put files in src directory. Files other than html are also possible (ignore).

### Step3

```bash
$ npm run start
```

or

```bash
$ node index.js
```
